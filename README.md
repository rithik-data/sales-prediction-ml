# 📊 Sales Prediction using Machine Learning

## 📌 Project Overview
This project predicts product sales based on advertising spend across different platforms like TV, Radio, and Newspaper.

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn

## 🤖 Model Used
- Linear Regression

## 📊 Features
- TV Advertising Budget
- Radio Advertising Budget
- Newspaper Advertising Budget

## 📈 Output
- Predicted Sales
- Model performance (MAE & R2 Score)

## 🚀 How to Run
1. Install dependencies:
   pip install pandas scikit-learn

2. Run:
   python analysis.py
